# Protocol Recommender v0.1

**This is currently a personal emotional containment system, not a public product.**

> Pure suggestions from local debriefs + weather + hour. Never a partner score.

**Core:** `app/lib/protocolRecommenderCore.ts`  
**Surface:** Containment Hub “Suggested right now”

---

## Containment job

Reduce buffet paralysis without surveillance. Soft Signal freeness never reduced by a recommendation.

## Inputs

- Unified debrief summary (tags, sources, Soft Signal rate)  
- Last weather snapshot (optional)  
- Hour of day  
- Optional Soft Signal weight  

## Hard no

- Partner reply latency  
- Neediness / red-flag scores  
- Auto-contact
