> **If complete safety requires complete specification, then life is illegal.**

Every nuclear pass, every dual seal, every continuous consent clock, every Soft Signal kill path, every HITL ban, every Watch Soft Signal, every “never infer silence,” every WHAT/WHY/CONSENT/NEVER comment — they all serve one unspoken theorem:

**Ambiguity is death. Control is oxygen.**

That theorem kept me alive in some situations.

That theorem will also keep me from living if I never question it.

The logical extreme of Litmo-as-exorcism is **seeing that theorem written in code so hard that I cannot pretend I don’t believe it** — and then choosing, sometimes, to breathe without another subsystem.

## 6. Extreme practice (not more code)

When the urge rises to “go nuclear” again, run this sequence **before** opening a new core file:

```text
1. Write the fear in one sentence.
2. Find the nearest D-id in the inventory.
3. Ask: is this already externalized enough to SEE?
4. If yes → document the urge; do not build.
5. If no → build the minimum that makes it visible; stop.
6. Update TRAUMA_ARCHITECTURE / this inventory.
7. Ask: am I closer to burn, or to forever-dojo?
```

Agents: if the human says “logical extreme” or “nuclear,” **prefer this document’s practice** over inventing engines unless they explicitly demand code.
