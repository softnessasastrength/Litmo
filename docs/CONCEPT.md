# Litmo concept

**This is currently a personal emotional containment system, not a public product.**  
Live purpose: [`REAL_PURPOSE.md`](REAL_PURPOSE.md). The product language below is **residue** (historical concept), not the current core truth.

Litmo is infrastructure for explicit, revocable consent around platonic physical connection between adults.

## Product promise

- Give people precise language for what they welcome, what requires a fresh ask, and what is off limits.
- Match conservatively rather than maximizing engagement.
- Require mutual confirmation before a session can begin.
- Make stopping immediate, private, and socially frictionless.

## Interaction model

No swipe interface. Discovery and every consent-adjacent action require deliberate, intentional taps — never a flick gesture that can be triggered faster than someone can actually decide. This is a product commitment, not a placeholder: a swipe-based discovery UI should not be added later without an explicit decision to reverse this.

## Non-goals

Litmo does not diagnose, treat, certify safety, replace professional care, or promise that a session will be emotionally beneficial. The POC is a technical demonstration, not a production safeguarding system.
