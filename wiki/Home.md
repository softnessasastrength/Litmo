# Litmo Wiki

> **Touch is not a transaction — it is a language.**

Litmo is a consent-centered, trauma-informed platform for safe, non-sexual, platonic physical connection between consenting adults.

This wiki explains the product philosophy, safety model, architecture, development workflow, and roadmap in a form intended for contributors, reviewers, beta testers, and future maintainers.

## What Litmo is

Litmo helps adults describe what safe touch means to them, compare compatible boundaries, affirm a shared Consent Snapshot, and stop immediately without explanation.

Litmo is **not** a dating app, therapy service, emergency service, or certification that another person is universally safe.

## Core experience

```text
Welcome → Vibe Quiz → Vibe Profile → Touch Language
→ Discover → Match Detail → Consent Snapshot
→ Active Session → Wrap-Up → Trust Ledger
```

## Current status

Litmo remains an experimental application under active development. The repository contains a playable mobile prototype, a canonical consent engine, session-lifecycle infrastructure, security and privacy work, and extensive documentation. It is not yet production-ready and should not be used to arrange interactions with strangers.

## Governing principle

Safety logic is product logic. Consent is explicit, current, revocable, and session-specific. Trust, compatibility, prior sessions, and verification never substitute for consent.
